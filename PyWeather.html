PyWeather Project

Summary

Visualizations

Technically, this project's main objective was the use of APIs, in this case, the OpenWeatherMap.org. Through the Python Script, it is possible to visualize the weather of 500+ cities across the world of varying distance from the equator with the usage of the Python Library MatPlotLib. This will help see the correlation between the nearness to the Equator and some Weather conditions.

This script was designed to obtain the following graphs:

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude

You can click the graphs in the Visualizations section.
